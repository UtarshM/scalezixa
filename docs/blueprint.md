# **App Name**: Scalezix.Academy

## Core Features:

- Homepage Overview: Provide a landing page with an overview of Scalezix Academy, its mission, and the AI-focused training programs offered.
- Detailed Curriculum Pages: Display detailed course pages that define curriculum for each AI program (e.g., AI Fundamentals, Machine Learning, Deep Learning, etc.) including modules, learning objectives, and instructor profiles.
- Image and Success Gallery: Gallery showcasing student projects, testimonials, and success stories to highlight the effectiveness of the training programs.
- FAQ Section: A FAQ section answering common questions about the academy, courses, enrollment, and other relevant information.
- Contact and Inquiry System: An interactive contact form and academy address to facilitate inquiries and communication with prospective students and visitors.
- AI Course Suggestion Tool: A tool that provides the courses which align with the school / college they are currently studying at

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to convey trust, intelligence, and professionalism, aligning with educational values.
- Background color: Light gray (#F0F2F5) providing a clean, neutral backdrop that enhances readability and focus.
- Accent color: Vibrant orange (#FF6B35) used sparingly to draw attention to key information or calls to action, adding energy.
- Body font: 'Inter', a sans-serif typeface, for body text to maintain a modern and highly readable appearance.
- Headline font: 'Space Grotesk', a sans-serif typeface, for headlines, which keeps a modern and technology-focused aesthetic.
- Use a consistent set of outline-style icons that visually represent the AI concepts taught in each course.
- Employ a grid-based layout to structure content logically across different pages, maintaining visual order and making navigation intuitive.
- Subtle transitions on page load and interactive elements (e.g., buttons, links) to enhance user experience without being distracting.
